函数部分

有界与最值定理，介值定理，平均值定理，零点定理

（开区间连续不一定存在最值）

函数在闭区间上连续，则函数在该区间存在最大值与最小值

一定存在x对应y

平均值一定在值域内

导数部分

费马定理

在点去可导，取极值，则该点导数为 0

罗尔定理

闭区间连续，开区间可导，两端值相等，则开区间内存在导数为 0

平均值定理，小区间存在大区间一定存在

拉格朗日中值定理

闭区间连续，开区间可导，存在函数值相减等于某一点导数乘以区间长度

柯西中值定理

​ 闭区间连续，开区间可导，函数导数不为 0

泰勒公式

拉格朗日是区间

佩亚诺是点
